# 机器学习（双语）

机器学习（双语）是人工智能学院在大三上学期开设的专业核心课程。

- 学习参考：

教科书（PRML，《机器学习与模式识别》） + 上课用 PPT

- 学习建议：

上课还是要认真听讲，部分知识涉及较为复杂的公式推导，需要掌握，不过考试的计算难度不会太高。

认真完成老师给的平时作业，使用 numpy 构建一些简单的机器学习程序，认真完成一定会有所收获。

- 考试题型：

22 级题型参考如下：

单选题 + 多选题 + 翻译题 + 解答题（8选5）+ 计算题
